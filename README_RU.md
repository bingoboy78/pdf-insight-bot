## Что это такое?
**PDF Insight Bot** — это готовый бессерверный пайплайн, который принимает PDF-файлы через удобный Web-интерфейс (или API), извлекает из них текст (используя OCR для картинок), разбивает на части и отправляет в большие языковые модели (LLM) для создания подробных конспектов, инсайтов и структурированного JSON-вывода.

Проект построен на платформе **Modal**, которая берет на себя всю работу с инфраструктурой: от тяжелой обработки PDF с помощью Tesseract OCR до предоставления API.

## Ключевые возможности
- **Web-интерфейс и API**: Загружайте файлы через простой браузерный интерфейс или автоматизируйте процесс через REST API.
- **Продвинутый OCR**: Автоматически распознает текст на отсканированных страницах с помощью `PyMuPDF` и `tesseract-ocr` (с поддержкой русского языка).
- **Поддержка разных LLM**: Работает с OpenAI, Anthropic, OpenRouter и NVIDIA NIM. Вы можете легко переключать модели без изменения кода.
- **Умное разбиение (Chunking)**: Длинные книги автоматически режутся на безопасные фрагменты, чтобы избежать переполнения контекстного окна (Context Window).
- **Асинхронность**: Тяжелые задачи выполняются в фоновом режиме на мощностях Modal, а результаты надежно сохраняются в Modal Volume.

## Как это работает?
1. **Загрузка**: Пользователь отправляет PDF через Web-интерфейс (FastAPI).
2. **Сохранение**: Файл и метаданные сохраняются в облачный диск (`Modal Volume`).
3. **Фоновая задача**: API запускает асинхронную обработку (`process_pdf_job`).
4. **Анализ**: 
   - Извлечение текста.
   - Распознавание картинок (OCR).
   - Нарезка на чанки.
5. **Синтез**: Чанки отправляются в выбранную нейросеть для получения конспекта на русском языке.
6. **Результат**: Выходные данные сохраняются в форматах `Markdown` и `JSON`.

## Быстрый старт

### 1. Требования
- Python 3.9+
- Аккаунт на [Modal.com](https://modal.com)

### 2. Установка Modal и Авторизация
```bash
pip install modal
modal setup
```

### 3. Настройка ключей (Секреты)
Проекту нужны ключи API для работы нейросети. Вам нужно создать безопасный "Секрет" в Modal с названием `pdf-insight-secrets`.

Выполните команду ниже в терминале, подставив свой API-ключ:
```bash
modal secret create pdf-insight-secrets \
  LLM_PROVIDER="nvidia" \
  LLM_API_KEY="nvapi-ВАШ_КЛЮЧ_ЗДЕСЬ" \
  LLM_MODEL="meta/llama-3.1-70b-instruct"
```
*(Поддерживаемые провайдеры: `openai`, `anthropic`, `openrouter`, `nvidia`)*

### 4. Деплой (Запуск) в облаке
Деплой приложения выполняется одной командой. Modal сам соберет Docker-контейнер, установит библиотеки и развернет API.

```bash
modal deploy -m src.app
```

После завершения команды, Modal выдаст вам ссылку (например, `https://<ваш-логин>--pdf-insight-bot-api.modal.run`). Откройте её в браузере, чтобы воспользоваться ботом!

## Использование через API

Если вы хотите интегрировать бота в свои автоматизации (например, через n8n или Telegram-бота), используйте следующие эндпоинты:

- `POST /submit` - Отправка PDF (ожидает `multipart/form-data` поле `file`). Возвращает `{"job_id": "...", "status": "processing"}`.
- `GET /status/{job_id}` - Проверка статуса.
- `GET /result/{job_id}` - Получение итогового JSON с инсайтами и тегами.
- `GET /summary/{job_id}` - Получение чистого Markdown конспекта.

## Безопасность 🔐
